## 项目介绍

这是一个工具链接导航站



## 安装步骤

1. 克隆仓库到本地：
    ```bash
    git clone git@github.com:mynxg/web-nav.git
    cd web-nav
    ```

2. 安装依赖：
    ```bash
    npm install
    ```


## 使用方法

1. 启动开发服务器：
    ```bash
    npm run dev
    ```

2. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)。



## Vercel 部署教程
登录 [Vercel](https://vercel.com/) 并创建一个新项目。

选择导入 GitHub 仓库，并选择你的项目仓库（web-nav）。


部署完成后，你可以在 Vercel 提供的域名上访问你的应用。
